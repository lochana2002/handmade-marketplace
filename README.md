


# Handmade Marketplace

handmade-marketplace using MERN Stack

A full-stack MERN marketplace where artisans can sell handmade products and customers can browse, purchase, and review them.

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Cloudinary
- Stripe

## Features

- Authentication
- Product Management
- Shopping Cart
- Orders
- Reviews
- Seller Dashboard
- Admin Dashboard
